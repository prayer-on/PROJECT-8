<h3>⚙️ Installation:</h3>
To run this project locally, you can follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/prayer-on/PROJECT-8.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm run dev
```
